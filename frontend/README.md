# Frontend App

React SPA for the Care Staff CPD Platform.

![React](https://img.shields.io/badge/React-19-61DAFB?style=flat-square&logo=react&logoColor=black)
![Vite](https://img.shields.io/badge/Vite-7-646CFF?style=flat-square&logo=vite&logoColor=white)

## Quick Start

```bash
npm install
npm run dev
```

The dev server proxies API requests to the backend. Ensure the backend is running and configured before testing authenticated features.
